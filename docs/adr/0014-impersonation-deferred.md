# ADR 0014 — Impersonation administrativa adiada

- Status: Aceito
- Data: 2026-07-16

## Decisão

Impersonation não será exposta nesta fase. Embora a modelagem possa evoluir para suportá-la, entregar o fluxo sem banner persistente, restrições de ações críticas, expiração curta, motivo e trilha dupla seria inseguro.

A capability permanece no roadmap e somente poderá ser implementada com todos os controles definidos no Prompt Mestre.
